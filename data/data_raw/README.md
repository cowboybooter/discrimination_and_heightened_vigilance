This directory contains all raw data that is external to the project.
This README should be updated to describe data sources.

[National Health Interview Survey](https://nhis.ipums.org/nhis/), via
IPUMS Health.

See NHIS 2023 Survey Description PDF

The data were usually conducted in-person, at 2,500 residences in each
geographically clustered areas, for a representative sample of the
United States. The data were conducted year-round, with a nationally
representative sample collected monthly. The data were self-reported by
the participants. Additionally, the data are generalizable to the
national level after correcting for aspects of research design,
including the strata and PSUs.

National Center for Health Statistics. National Health Interview Survey,
2023 survey description. 2024. Available from:
https://ftp.cdc.gov/pub/Health_Statistics/NCHS/Dataset_Documentation/NHIS/2023/srvydesc-508.pdf
